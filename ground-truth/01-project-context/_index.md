## One-paragraph framing

Rootstrap is repositioning from a *nearshore software agency* (staff augmentation + product
studio) to an **AI-native capability delivery partner** whose durable asset is **Ground Truth**.
The new website must tell that story *and be built by the method it describes* — including a
**Ground Truth Explorer** that lets visitors experience the model. This website is also the
**pilot** for the Ground Truth Blueprint we intend to run on every future project.
