# Capability: How we work (the method)

> Human-facing unit of value. Executable detail in the user stories below. This is the page that
> makes the agentic method *believable* — and proves Rootstrap practices it (this very site is the dogfood).

## Purpose
Make the AI-native delivery method tangible and credible — Knowledge → Ground Truth → Specs →
Agents → Software, run as reusable loops with human gates and evals — and show that Rootstrap
actually works this way (not AI theater).
